Workspace working instructions¶

There is a master workspace with all the needed package installed

The master workspace is in /home/pi/rosbotics_ws/master_ws

Copy to the Raspberry Pi your src workspace:

your_workspace/src

Your workspace must only include the src folder

Open a terminal shell

Change to the your workspace folder:

cd path_to_workspace_folder

Source the master workspace:

source /home/pi/rosbotics_ws/master_ws

Compile the workspace 1

catkin_make

To use your workspace:

source your_workspace/devel/setup.bash

The nodes that provide the different topics are:

Provides robot model, odometry:

roslaunch gopigo3_node gopigo3.launch

Provides laser scans:

roslaunch ydlidar lidar.launch (or roslaunch rplidar_ros rplidar.launch)

Provides camera images:

roslaunch raspicam_node camerav2_1280x960_10fps.launch enable_raw:=true camera_frame_id:="laser_frame"

Footnotes

1

To run your scripts in the real robot, no references to Gazebo must be done.
